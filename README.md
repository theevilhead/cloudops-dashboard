## How to run

1. Clone the repository:
2. Install dependencies

  ```
    npm install
  ```

3. To start the application in development mode:

  ```
    npm run dev
  ```

4. To build the application for production:

  ```
    npm run build
  ```

5. To start the application in production mode:

  ```
    npm run start
  ```
